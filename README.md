# 🌿 KeenKeeper

**Your personal shelf of meaningful connections.** KeenKeeper helps you browse, tend, and nurture the relationships that matter most. In a digital world where it's easy to lose touch, KeenKeeper acts as your personal relationship manager to ensure no friend is forgotten.

## ✨ Key Features

* **🎯 Intelligent Relationship Goals:** Set custom contact frequencies for every friend (e.g., "Connect every 30 days"). The system automatically tracks days since your last contact and alerts you when a relationship is "Overdue" or "Almost Due."
* **📊 Friendship Analytics:** Visualize your social habits with a dedicated stats dashboard. Understand your interaction patterns—whether you prefer calling, texting, or meeting up—through intuitive doughnut charts and data summaries.
* **⏳ Interactive Interaction Timeline:** Keep a chronological record of every meaningful moment. From quick texts to industry conferences, the timeline ensures you always remember the context of your last conversation.

## 📂 Project Structure

- **Dashboard:** A bird's eye view of all your friends with status badges (On-Track, Overdue, Almost Due).
- **Friend Profile:** Deep dive into specific relationships with "Quick Check-In" actions (Call, Text, Video).
- **Timeline:** A unified feed of all interactions across your entire network.
- **Stats:** Visual breakdown of how you spend your social energy.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ab-siddik-ru-cse/keen-keeper 
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```


2. **Run the development server:**

   ```bash
   npm run dev
   ```
